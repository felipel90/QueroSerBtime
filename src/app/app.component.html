DojoPuzzles - Analizando urls

See more.

Dada uma URL, desenvolva um programa que indique se a URL é válida ou não e, caso seja válida, retorne as suas partes constituintes.

Exemplo:

http: http://www.google.com/mail?user=fulano

ssh: ssh://fulano%senha@git.com/

Teste:

Insira uma url HTTP/SSH

result:

URL inválida
